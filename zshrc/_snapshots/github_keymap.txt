Keymap: github_keymap.zsh

  `   1   2   3   4   5   |   6   7   8   9   0   [   ] 
      '   ,   .  <p>  y   |   f  (g) <c> (r)  l   /   =   \ 
     <a> (o)  e  <u>  i   |  <d> (h) (t) <n>  s   - 
      ;   q   j   k   x   |  <b>  m   w   v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ h                         # Show this keymap
  $ h <regex>                 # Search this keymap

  $ h.<key>                   # This mapping takes no variable
  $ h.<key> <var>             # This mapping takes one variable
  $ h.<key> <var>?            # This mapping takes zero or one variable
  $ h.<key> <var>*            # This mapping takes zero or multiple variables
  $ h.<key> (1-10)            # This mapping takes an exact value from the list

     ^                        # The `.` is only for documentation
                              # Omit it when invoking a mapping

Keymap List

  $ h.h <match>* <-mismatch>* # Go to GitHub & list repos & filter
  $ h.t                       # Go to repo in pasteboard
  $ h.tt                      # Copy current repo to pasteboard

  $ h.a                       # Open current repo in GitHub Desktop
  $ h.n                       # Create new PR & open it
  $ (|)? h.g                  # Create new gist & open it
  $ h.gg                      # Open tab to create gist

  $ h <repo>                  # Open specified repo
  $ h.o                       # Open current repo
  $ h.p <pr>?                 # Open PRs (or a specific PR)
  $ h.c <sha>?                # Open a commit (Default: Pasteboard)

  $ h.r <match>* <-mismatch>* # List remote repos & filter
  $ h.rr                      # Save copy of remote repos

  $ h.url                     # Remote url
  $ h.domain                  # Remote domain
  $ h.org                     # Org name
  $ h.repo                    # Repo name
  $ h.branch                  # Branch name
