Keymap: docker_keymap.zsh

  `   1   2   3   4   5   |   6   7   8   9   0   [   ] 
      '   ,   .   p   y   |   f   g   c   r   l   /   =   \ 
      a   o   e   u   i   |  <d>  h   t   n  <s>  - 
      ;   q   j   k   x   |   b   m   w   v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ d                    # Show this keymap
  $ d <regex>            # Search this keymap

  $ d.<key1>             # This mapping takes no argument
  $ (|)? d.<key2>        # This mapping can be invoked after a `|`
  $ d.<key3> <var>?      # This mapping takes an optional variable
  $ d.<key4> <var> (val) # This mapping takes a variable and an exact value

     ^                   # The `.` is only for documentation
                         # Omit it when invoking a mapping

  $ d <docker command>
