Keymap: docker_keymap.zsh

  `   1   2   3   4   5   |   6   7   8   9   0   [   ] 
      '   ,   .   p   y   |   f   g   c   r   l   /   =   \ 
      a   o   e   u   i   |  <d>  h   t   n  <s>  - 
      ;   q   j   k   x   |   b   m   w   v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ d                  # Show this keymap
  $ d <regex>          # Search this keymap

  $ d.<key>            # This mapping takes no variable
  $ d.<key> <var>      # This mapping takes one variable
  $ d.<key> <var>?     # This mapping takes zero or one variable
  $ d.<key> <var>*     # This mapping takes zero or multiple variables
  $ d.<key> (1-10)     # This mapping takes an exact value from the list
  $ (|)? d.<key>       # This mapping can be invoked after a `|`

     ^                 # The `.` is only for documentation
                       # Omit it when invoking a mapping

Keymap List

  $ d <docker command> 

  $ d.s                # Login with AWS credentials
