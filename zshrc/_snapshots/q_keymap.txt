Keymap: q_keymap.zsh

  `   1   2   3  <4>  5   |   6   7   8   9  <0>  [   ] 
      '   ,   .  <p>  y   |   f   g   c   r   l   /   =   \ 
      a   o   e   u   i   |   d  <h>  t   n   s   - 
      ;  <q> <j>  k   x   |   b   m   w   v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ q              # Show this help
  $ q <regex>      # Filter key mappings

  $ q.<key>        # Invoke a <key> mapping
  $ q.<key> <arg>  # Invoke a <key> mapping with <arg>

     ^             # The `.` is only for documentation
                   # Omit it when invoking a mapping

Keymap List

  $ q.q            # Chat without MCP
  $ q.h            # Chat with GitHub MCP, and `pbcopy` trust instructions
  $ q.j            # Chat with Jira MCP, and `pbcopy` trust instructions

  $ q.0 <command>? # Invoke `q`
  $ q.4 <command>? # Invoke `q chat` with `claude-4-sonnet`

  $ q.p            # Push `amazonq` folder to `scratch` repo
  $ q.P            # Pull `amazonq` folder from `scratch` repo
