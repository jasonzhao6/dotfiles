Keymap: q_keymap.zsh

  `   1   2   3  <4>  5   |   6   7   8   9  <0>  [   ] 
      '   ,   .  <p>  y   |   f   g  <c>  r   l   /   =   \ 
     <a> <o>  e   u   i   |  <d> <h>  t   n  <s>  - 
      ;  <q> <j> <k>  x   |   b  <m>  w   v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ q              # Show this keymap
  $ q <regex>      # Search this keymap

  $ q.<key>        # This mapping takes no variable
  $ q.<key> <var>  # This mapping takes one variable
  $ q.<key> <var>? # This mapping takes zero or one variable
  $ q.<key> <var>* # This mapping takes zero or multiple variables
  $ q.<key> (1-10) # This mapping takes an exact value from the list

     ^             # The `.` is only for documentation
                   # Omit it when invoking a mapping

Keymap List

  $ q.q            # Chat without MCP
  $ q.a            # Chat with Atlassian MCP
  $ q.d            # Chat with Datadog MCP
  $ q.h            # Chat with GitHub MCP
  $ q.s            # Chat with SDLC MCP

  $ q.c            # Vibe code in the current repo
  $ q.j            # Vibe code in the JCard repo
  $ q.k            # Vibe code in the K8s Helm repo

  $ q.0 <command>? # Invoke `q`
  $ q.4 <command>? # Invoke `q chat` with `claude-sonnet-4.5`

  $ q.o            # Open `amazonq` folder in Finder
  $ q.m            # Edit `amazonq` folder in TextMate
  $ q.p            # Push `amazonq` folder to `scratch` repo
  $ q.P            # Pull `amazonq` folder from `scratch` repo
