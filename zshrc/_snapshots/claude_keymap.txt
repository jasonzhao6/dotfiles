Keymap: claude_keymap.zsh

  `   1   2   3   4   5   |   6   7   8   9   0   [   ] 
      '   ,   .   p   y   |   f   g  <c> <r> <l>  /   =   \ 
      a   o   e  <u>  i   |   d   h   t   n   s   - 
      ;   q   j   k   x   |   b  <m>  w   v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ c              # Show this keymap
  $ c <regex>      # Search this keymap

  $ c.<key>        # This mapping takes no variable
  $ c.<key> <var>  # This mapping takes one variable
  $ c.<key> <var>? # This mapping takes zero or one variable
  $ c.<key> <var>* # This mapping takes zero or multiple variables
  $ c.<key> (1-10) # This mapping takes an exact value from the list

     ^             # The `.` is only for documentation
                   # Omit it when invoking a mapping

Keymap List

  $ c.c            # Start new session
  $ c.r            # Continue last session
  $ c.l <match>?   # Resume matching session
  $ c.m            # Edit config folder in TextMate
  $ c.u            # Push config to `scratch` repo (not `p` b/c `cp` is reserved)
  $ c.U            # Pull config from `scratch` repo
