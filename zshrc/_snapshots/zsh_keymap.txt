Keymap: zsh_keymap.zsh

  `  <1> <2>  3   4   5   |   6   7   8   9  <0>  [   ] 
      '   ,   .  <p>  y   |  <f>  g   c   r   l   /   =   \ 
     <a>  o  <e>  u   i   |   d  (h) <t>  n  <s>  - 
      ;   q   j   k   x   |   b  (m) <w>  v  (z)

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ z                         # Show this keymap
  $ z <regex>                 # Search this keymap

  $ z.<key>                   # This mapping takes no variable
  $ z.<key> <var>             # This mapping takes one variable
  $ z.<key> <var>?            # This mapping takes zero or one variable
  $ z.<key> <var>*            # This mapping takes zero or multiple variables
  $ z.<key> (1-10)            # This mapping takes an exact value from the list

     ^                        # The `.` is only for documentation
                              # Omit it when invoking a mapping

Keymap List

  $ z <markdown file>         # Clear screen and render with `mdcat`
  $ z.z <markdown file>       # Render with `mdcat`

  $ z.e                       # Edit in IntelliJ
  $ z.m                       # Edit in TextMate
  $ z.mm                      # Edit secrets in TextMate
  $ z.s                       # Source
  $ z.t <match OR 1-5>?       # Run tests by name or section #

  $ z.w <name>                # Custom `which` lookup
  $ z.a <match>* <-mismatch>* # List aliases & filter
  $ z.f <match>* <-mismatch>* # List functions & filter

  $ z.h <grep>?               # List history
  $ z.hc                      # Clear history
  $ z.hm                      # Edit history file in TextMate

  $ z.1                       # No session history
  $ z.2                       # Session history in memory
  $ z.0                       # Session history in memory & file

  $ z.p                       # Push other dotfiles to this repo
  $ z.P                       # Pull other dotfiles from this repo
