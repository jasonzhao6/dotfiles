Keymap: other_keymap.zsh

  `  (1) (2)  3   4   5   |   6   7  <8>  9  <0>  [   ] 
      '   ,   .  <p> <y>  |  <f>  g  (c) <r>  l   /   =   \ 
     <a> (o) <e> (u) (i)  |  (d)  h  <t> <n> <s>  - 
      ;  (q) <j> (k) (x)  |  <b> <m> <w>  v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ o                             # Show this keymap
  $ o <regex>                     # Search this keymap

  $ o.<key>                       # This mapping takes no variable
  $ o.<key> <var>                 # This mapping takes one variable
  $ o.<key> <var>?                # This mapping takes zero or one variable
  $ o.<key> <var>*                # This mapping takes zero or multiple variables
  $ o.<key> (1-10)                # This mapping takes an exact value from the list

     ^                            # The `.` is only for documentation
                                  # Omit it when invoking a mapping

Keymap List

  $ o <url>                       # Open specified url
  $ o <path>                      # Open specified path in Finder

  $ o.o                           # Open current directory in Finder
  $ o.e <path>?                   # Open IntelliJ (Default: Current dir)
  $ o.m <path>?                   # Open TextMate (Default: Current dir)

  $ o.s                           # Sleep
  $ o.a                           # Stay awake
  $ o.n                           # Stay on task

  $ o.c                           # Copy last output
  $ o.cc                          # Copy last command
  $ o.y                           # Alias for `pbcopy`
  $ o.p                           # Alias for `pbpaste`

  $ o.k                           # Clear terminal
  $ o.kk                          # Show archived terminal outputs
  $ o.kc                          # Clear archived terminal outputs

  $ o.12                          # Open diff files in TextMate
  $ (|)? o.1                      # Save pasteboard value to `diff.1.txt`
  $ (|)? o.2                      # Save pasteboard value to `diff.2.txt`
  $ o.11                          # Save last output to `diff.1.txt`
  $ o.22                          # Save last output to `diff.2.txt`
  $ o.0                           # Empty `diff.1.txt` and `diff.2.txt`
  $ o.u <file 1>? <file 2>?       # Unified diff
  $ o.uu <file 1>? <file 2>?      # Side by side diff

  $ o.t <command>                 # Time command execution
  $ o.w <seconds>? <command>      # Loop command with sleep (Default: 1s)
  $ o.f <start> <finish> (~~)     # Run command sequence in foreground
  $ o.b <start> <finish> (~~)     # Run command sequence in background

  $ (|)? o.i <file>? <col index>? # CSV: Sort lines by column index
  $ (|)? o.ii <file>? <i1> <i2>?  # CSV: Swap columns by indexes
  $ o.x <file 1>? <file 2>?       # CSV: Keep lines matching 1st column
  $ o.xx <file 1>? <file 2>?      # CSV: Drop lines matching 1st column

  $ o.8                           # Use Java 8
  $ o.d <url>                     # DNS dig
  $ o.df                          # DNS flush
  $ o.j <url> <regex> <num lines> # Curl json endpoint
  $ o.q                           # Format sql query from stdin
  $ o.q '<sql>'                   # Format sql query from cli arg
  $ o.r <before> <after>          # Rename files in current directory
