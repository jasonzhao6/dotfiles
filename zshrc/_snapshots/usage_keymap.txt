Keymap: usage_keymap.zsh

  `   1   2   3   4   5   |   6   7   8   9   0   [   ] 
      '   ,   .   p   y   |   f   g  <c>  r   l   /   =   \ 
     <a>  o   e  <u>  i   |  <d> <h> (t) <n>  s   - 
      ;   q   j   k   x   |   b  <m>  w   v   z 

  `<>` key initials have one mapping
  `()` key initials have multiple mappings

Keymap Usage

  $ u                 # Show this keymap
  $ u <regex>         # Search this keymap

  $ u.<key>           # This mapping takes no variable
  $ u.<key> <var>     # This mapping takes one variable
  $ u.<key> <var>?    # This mapping takes zero or one variable
  $ u.<key> <var>*    # This mapping takes zero or multiple variables
  $ u.<key> (1-10)    # This mapping takes an exact value from the list

     ^                # The `.` is only for documentation
                      # Omit it when invoking a mapping

Keymap List

  $ u.u <n>?          # Usage overview, for last <n> calendar days (Default: All)
  $ u.d <n>?          # Usage by day of week, for last <n> calendar days (Default: All)
  $ u.h <n>?          # Usage by hour of day, for last <n> calendar days (Default: All)

  $ u.n <n OR match>? # Top namespaces, for last <n> calendar days (Default: All)
  $ u.a <n OR match>? # Top aliases, for last <n> calendar days (Default: All)

  $ u.m               # Open usage file in TextMate
  $ u.c               # Clear usage file

  $ u.t <n>           # Back up real data, generate test data for <n> calendar days
  $ u.tt              # Restore real data from backup
